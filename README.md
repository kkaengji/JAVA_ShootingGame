# JAVA_ShootingGame
 JAVA를 이용하여 만든 포켓몬 ShootingGame APP 입니다.

## 프로젝트 정보
### 설치
- 사이즈 등 수정 시 JAVA Eclipse
- Framework.apk를 안드로이드 기반 스마트폰에서 설치

### 사용 방법
- apk를 설치 하였으면 실행
- 점수 아이템을 먹으면 점수가 UP
- 생명 아이템을 먹으면 LIFE가 UP
- 파워 아이템을 먹으면 공격력이 UP
- LIFE 가 0이 되면 게임이 종료됩니다.
